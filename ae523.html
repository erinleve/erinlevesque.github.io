AEROSP 523: Computational Fluid Dyanmics

Course Description

AEROSP 523 is a graduate-level course that introduces students to fundamental knowledge of computational fluid dynamics (CFD). This course tasks students to write custom code that numerically computes solutions to partial differential equations. Topics covered include: one and two-dimensional boundary value problems, simple iterative methods, multigrid methods, stability analysis, and finite volume schemes for conservation laws. It is the successor to AEROSP 423 Numerical Methods course, which I also completed.
